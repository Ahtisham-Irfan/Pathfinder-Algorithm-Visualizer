# Pathfinder — Algorithm Visualizer

An interactive, animated visualizer for four classic pathfinding algorithms — built with vanilla HTML, CSS, and JavaScript. No frameworks, no libraries.

## Features
- **Algorithms**: A* Search, Dijkstra, Breadth-First Search, Depth-First Search
- Draw walls by click-and-drag directly on the grid
- Drag the start and end nodes anywhere on the board
- One-click random maze generation
- Animated exploration: watch each algorithm's search frontier expand in real time, then see the shortest path highlighted
- Live stats: nodes explored, path length, and computation time
- Adjustable animation speed (fast / medium / slow)
- Fully responsive, keyboard-accessible, respects `prefers-reduced-motion`

## Why this is a good portfolio piece
It demonstrates real algorithm implementation (not just calling a library), state management, DOM manipulation, and animation timing — all in dependency-free JavaScript.

## Files
- `index.html` — page structure and controls
- `style.css` — all styling and animations
- `script.js` — grid logic, all four algorithms, and animation engine

## Run locally
Just open `index.html` in a browser.
